Attendance insights

📌 Overview

This project focuses on analyzing student attendance and performance data using data mining techniques. It involves data preprocessing, cleaning, transformation, and preparation for applying machine learning algorithms in WEKA.

The project demonstrates a complete data mining pipeline including dataset preparation, normalization, duplicate removal, and generation of ARFF files for further analysis.

⸻

🎯 Objectives • Clean and preprocess raw attendance data • Handle missing values and inconsistencies • Generate multiple dataset versions (original, normalized, duplicate-free) • Prepare datasets for WEKA algorithms • Apply data mining techniques such as classification, clustering, and association rule mining
